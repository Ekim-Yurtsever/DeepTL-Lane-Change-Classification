# Mask_RCNN_Lane_Change

original repo: https://github.com/matterport/Mask_RCNN

1- download weights, mask_rcnn_coco.h5, to the main folder. Link to the weigts: (https://github.com/matterport/Mask_RCNN/releases)

2- Run detect_objects.py to create images with segmentation masks.

Versions and Requirements:
- python 3.6.6 
- tensorflow 1.9.0
- cython 0.28.5
- keras 2.2.2
- IPython 7.3.0
- pycocotools 2.0.0
- scikit-image 0.14.2

Note that to use Yolo you have to download the weights: https://pjreddie.com/media/files/yolov3-spp.weights
